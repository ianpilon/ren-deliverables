📅 February 23, 2026 — Sunday
Voice calls FIXED! Found the Twilio WebSocket root cause.
✅ Milestone
▶
🎉 Voice Calls FIXED!
Root cause: Twilio doesn't forward URL query params in WebSocket connections.
🔧 Technical Fix
Auth token in ?token=... was being stripped, causing stream rejection. FIX: Patched OpenClaw voice-call extension: 1. twilio.ts → Use <Parameter name="token"> in TwiML instead of URL query string 2. media-stream.ts → Extract token from customParameters instead of URL Cleared jiti cache to force recompile. TTS and STT both working!
